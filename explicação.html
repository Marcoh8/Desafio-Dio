Explicação: HTML: Cria uma interface simples com campos de entrada para vitórias e derrotas, um botão para calcular, e um parágrafo para exibir o resultado. JavaScript: Define a função calcularSaldo e a função exibirResultado que é chamada quando o botão é clicado. A função exibirResultado obtém os valores dos campos de entrada, chama calcularSaldo, e exibe o resultado no parágrafo com id resultado.
